## นิยามจากลูกค้า (ZD)

| รหัส SAP (ประชุม) | ความหมาย (ลูกค้า) |
|-------------------|-------------------|
| **ZD01** | Breakdown / เครื่องหยุด |
| **ZD02** | Preventive Maintenance (PM) + Maintenance Code ข้างหน้า |
| **ZD05** | General Repair |

---

## แมปกับไฟล์ IW37N ในโรงงาน (ZB)

คอลัมน์ `Type` / `wktype` หลัง import ยังเป็น **ZB*** — ระบบแมปดังนี้เพื่อ filter และแสดงผลให้ตรงภาษาลูกค้าในประชุม:

| ZB ใน IW37N / DB (กรอง) | ZD (ประชุมลูกค้า) | ความหมาย ZD |
|-------------------------|-------------------|-------------|
| **ZB05** | **ZD01** | Breakdown / เครื่องหยุด |
| **ZB02** | **ZD02** | Preventive Maintenance (PM) |
| **ZB01** | **ZD05** | General Repair |

รหัสอื่น (PM01–04, ZB03, ZB04, …) ไม่ได้กำหนด ZD ในประชุม — แสดงตาม master / ค่าในไฟล์

**หมายเหตุประชุม:** ZD02 กล่าวถึงไฟล์เก่าที่มี **ZB02/ZB05** — ในแอปกรอง PM หลักด้วย **ZB02**; Breakdown แยก **ZB05 → ZD01**

## แสดงในแอป

- Filter **Type (ZB / SAP ZD)** — ป้าย `ZB02 → ZD02 Preventive Maintenance (PM)`
- คอลัมน์ Type: `ZB02 · ZD02`
- กล่องสรุปใต้ตัวกรอง (อิงประชุมลูกค้า)
